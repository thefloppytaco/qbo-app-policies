Privacy Policy

Effective date: 2026-05-07

This Privacy Policy describes how the QuickBooks Online integration application (the "Application") provided by Moshe Baum (the "Provider") handles information.

1. Information We Access

When the User authorizes the Application via OAuth, the Application receives access tokens that allow it to read and modify QuickBooks Online data. This may include financial records, customer and vendor information, invoices, bills, transactions, payroll data, and company metadata.

2. How Information Is Used

The Application uses QuickBooks data solely to perform the operations the User initiates (for example, retrieving a balance sheet, creating an invoice, or running a financial report). Data is processed in real time and is not retained by the Application beyond the duration of the user's session.

3. Information We Do Not Collect

The Provider does not operate any backend server that stores QuickBooks data. The Application runs locally on the User's device. The Provider does not collect analytics, telemetry, or usage data. No QuickBooks data is shared with any third party.

4. Storage of Credentials

OAuth tokens are stored locally on the User's device, in configuration files protected by standard operating-system file permissions. The User is responsible for the security of their device and credentials.

5. Third-Party Services

The Application communicates only with Intuit's QuickBooks Online API. It does not transmit data to any other third-party service. Intuit's own privacy practices apply to data stored within QuickBooks Online and are governed by Intuit's Privacy Statement.

6. Data Retention and Deletion

Because the Provider does not store User data, there is nothing to retain or delete on the Provider's side. To revoke access, the User can disconnect the Application from their QuickBooks Online account at any time via QuickBooks Online → Settings → Apps.

7. Children's Privacy

The Application is not directed to children under 13 and does not knowingly collect data from children.

8. Changes to This Policy

The Provider may update this Privacy Policy. Updates will be posted at the same URL with a revised effective date.
